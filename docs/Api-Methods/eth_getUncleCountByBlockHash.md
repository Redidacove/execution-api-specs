# eth_getUncleCountByBlockHash

Returns the number of uncles in a block from a block matching the given block hash

## Parameter Names

### Name - Block hash
#### `schema`
#### title - `32 byte hex value`
###### type - `string`

## Result

### Name - Uncle count

#### schema 
`One Of`
##### title - `Not Found`
##### type - `null`

##### title - `Uncle count`
##### type - `string`
